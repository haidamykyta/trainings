# Неделя 1 — Входная (найди рабочие веса)

**Задача:** Не рекорды. Найди правильный вес в каждом упражнении при диапазоне **12-20 повторений**.
**RIR этой недели:** 2 везде. Последний подход изоляции — 1 RIR.
**Кардио:** Велосипед или AirBike (не наклонная дорожка).

> Разминка по [RULES.md](RULES.md) перед каждым днём — обязательно.
> Кор-блок (планка + Dead bug) — после каждого дня 5-7 мин → [BACK_HEALTH.md](BACK_HEALTH.md)

### ПРЕДПЛЕЧЬЯ (новый блок — для объёма и вен)

| Упражнение | Подходы x Повт | Отдых | Примечание |
|-----------|----------------|-------|------------|
| Сгибание запястий со штангой / гантелями | 4 x 15-20 | 45 сек | Хват снизу. Только кисть двигается, предплечье на скамье |
| Разгибание запястий обратным хватом | 3 x 15-20 | 45 сек | Хват сверху. Медленно, контроль |
| Обратные сгибания (Reverse curl) с EZ-штангой | 3 x 15-20 | 60 сек | Хват сверху — брахиалис + брахиорадиалис (главная мышца предплечья) |
| Молотки (hammer curl) | 2 x 15-20 | 45 сек | Нейтральный хват — работает брахиорадиалис активно |
| Farmer's walk (фермерская ходьба) | 3 x 30-40 м | 60 сек | Максимально тяжёлые гантели которые можешь держать |

> **Почему Farmer's walk:** Изометрическое удержание тяжёлого веса = самое эффективное упражнение для толщины предплечий. Плюс grip strength — это накачивает сами сосуды, что делает вены более заметными.

> **Для видимости вен:** Вены выперают при низком % жира + накаченных предплечьях. Предплечья накачиваются медленнее всего (много медленносокращающихся волокон) — нужна стабильная работа. 15-20 повт = оптимально.
